Selectores de atributos en CSS

  1. [att]: Este selector selecciona todos los elementos que tienen un atributo específico, sin importar su valor. Es útil cuando deseas aplicar un estilo a todos los elementos que tienen un cierto atributo, independientemente de su valor. Por ejemplo, si deseas aplicar un estilo a todos los elementos que tienen el atributo `data-toggle`, puedes hacerlo de la siguiente manera:

    
    [data-toggle] {
      /* Estilos para elementos con el atributo data-toggle */
    }

    Situación real: Puedes usar este selector para aplicar estilos a todos los botones en una página web que tienen atributos personalizados para manejar interacciones específicas, como menús desplegables o ventanas modales.

  2. [att=val]: Este selector selecciona todos los elementos que tienen un atributo específico con un valor específico. Es útil cuando deseas aplicar un estilo solo a elementos que tengan un valor de atributo específico. Por ejemplo, si deseas aplicar un estilo solo a los elementos con el atributo `type` con valor `button`, puedes hacerlo de la siguiente manera:

    
      [type="button"] {
        /* Estilos para elementos con type="button" */
      }

    Situación real: Puedes usar este selector para aplicar estilos solo a los botones en un formulario, lo que ayuda a mejorar la apariencia y la usabilidad de los botones.

  3. [att~=val]: Este selector selecciona elementos cuyo atributo contiene un valor específico como una palabra completa o parcial, separada por espacios. Es útil cuando deseas seleccionar elementos que tengan ciertas palabras clave en un atributo. Por ejemplo, si deseas aplicar un estilo a todos los elementos con la clase `my-class` en una lista de clases separadas por espacios, puedes hacerlo de la siguiente manera:

    
      [class~="my-class"] {
        /* Estilos para elementos con la clase "my-class" en su lista de clases */
      }

    Situación real: Puedes usar este selector para aplicar estilos a elementos que comparten una palabra clave específica en su atributo de clase, como resaltar todos los elementos con la clase error en un formulario de validación.

  4. [att|=val]: Este selector selecciona elementos cuyo atributo tiene un valor que coincide con el valor especificado o comienza con el valor especificado seguido de un guión (-). Es útil cuando deseas seleccionar elementos con atributos de idioma, como los atributos lang. Por ejemplo, si deseas aplicar un estilo a todos los elementos con el atributo lang igual a "en" o que comience con "en-", puedes hacerlo de la siguiente manera:

          
      [lang|="en"] {
          /* Estilos para elementos con el atributo lang igual a "en" o que comienza con "en-" */
      }

    Situación real: Puedes usar este selector para aplicar estilos a elementos de un sitio web en un idioma específico o subidioma, como resaltar elementos en inglés en una página multilingüe.
