Enlaces internos en HTML

1. Añadir un id único al elemento que queremos enlazar

    <p id="parrafo-1">

2. Crear el enlace hacia el id

    <a href="#parrafo-1">Párrafo 1

    Observa que el atributo href en un enlace interno se forma con el caracter almohadilla más el id del elemento X.
  
    TIP:
       
      Sirve para crear un enlace que NO va a sitio alguno y se conocen como enlaces muertos o dead link
